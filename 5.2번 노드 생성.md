# 5. 2번 노드 생성

- **1번 노드의 VM 복제**

vboxmanage를 이용해 19crac1.vdi로부터 19crac2.vdi를 복제한다.

```
E:\oraclevm\19crac\19crac1>vboxmanage clonemedium disk ./19crac1.vdi ../19crac2/19crac2.vdi
```
